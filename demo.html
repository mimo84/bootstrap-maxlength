Bootstrap-maxlength

This plugin integrates by default with Twitter bootstrap using badges to display the maximum lenght of the field where the user is inserting text. Uses the HTML5 attribute "maxlength" to work.

Download this project as a .zip file Download this project as a tar.gz file

Bootstrap MaxLength

Demo

Default usage

$('input[maxlength]').maxlength();

Change the threshold value

The plugin will show the indicator for remaining chars since the number of chars left are 20.

$('input.className').maxlength({
    threshold: 20
});

Few options 

$('input.className').maxlength({
            alwaysShow: true,
            threshold: 10,
            warningClass: "label label-success",
            limitReachedClass: "label label-important"
        });

We can also set some text in at the beginning and at the end of the idicator. The separator can be also customized:

$('input.className').maxlength({
            alwaysShow: true,
            threshold: 10,
            warningClass: "label label-success",
            limitReachedClass: "label label-important"
        });
